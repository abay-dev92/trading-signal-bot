# Crypto Signal Bot

Python bot that fetches Binance market data, calculates EMA and RSI indicators, and sends trading alerts to Telegram.

## Features
- Binance OHLCV data
- EMA 9 / EMA 21
- RSI 14
- BUY / SELL signal generation
- Telegram notifications

## Stack
- Python
- CCXT
- Pandas
- Requests
- Telegram Bot API

## Installation

pip install -r requirements.txt
